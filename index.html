Master Thesis (WOP)

The title of the master's thesis is "Physics-based Simulation of Autonomous Bubble Detection Based on Visual and Acoustic Sensing", it is written at the MARUM. The thesis involves the development of a simulation of an underwater environment in Unreal Engine 5.3. The simulation encompasses the simulation of sensors such as a sonar and a camera, as well as an underwater environment with methane leaks. Additionally, a system for autonomous bubble detection is developed. The system utilises two U-Net models to segment sonar and camera images provided by the simulation. (For more informations see the extended abstract (its still work in progress))

Master Arbeit (WOP)

Der Titel der Masterarbeit lautet „Physics-based Simulation of Autonomous Bubble Detection Based on Visual and Acoustic Sensing“, sie wird beim MARUM geschrieben. Die Arbeit beinhaltet die Entwicklung einer Simulation einer Unterwasserumgebung in der Unreal Engine 5.3. Die Simulation umfasst die Simulation von Sensoren wie einem Sonar und einer Kamera, sowie einer Unterwasserumgebung mit Methanlecks. Zusätzlich wird ein System zur autonomen Erkennung von Blasen entwickelt. Das System nutzt zwei U-Netz-Modelle, um die von der Simulation gelieferten Sonar- und Kamerabilder zu segmentieren. (Weitere Informationen finden Sie im extended abstract (welcher noch in work in progress ist))

Master Project (Triplesim)

The Triplesim master project of CGVR and MARUM works towards creating a simulation of an underwater exploration of an icy moon in a virtual environment. This project takes place in the context of the TRIPLE project, which plans such missions in the future. A focus of this project is simulating the navigation of an autonomous underwater vehicle (AUV) in Unreal Engine 5.2. For this purpose, the simulation contains many sensors that are commonly used for navigation.

Master Projekt (Triplesim)

Das Masterprojekt Triplesim von CGVR und MARUM beschäftigt sich mit der Simulation einer Unterwassererkundung von Eismonden in einer virtuellen Umgebung. Dieses Projekt findet im Rahmen des TRIPLE-Projekts statt, welches solche Missionen in Zukunft plant. Ein Schwerpunkt dieses Projekts ist die Simulation der Navigation eines autonomen Unterwasserfahrzeugs (AUV) in der Unreal Engine 5.2. Zu diesem Zweck enthält die Simulation diverse Sensoren, die üblicherweise zur Navigation verwendet werden.

Human–computer interaction course project

Master course of the Human Computer Interaction group at the university of bremen. The idea of this project was to develop a smart home system that uses tangible objects as a method of expressing emotions. It uses 3D printed objects with RFID chips and an RFID scanner, based on the chosen object the system changes music and lighting.

Mensch-Computer-Interaktion Kurs projekt

Master kurs der Human Computer Interaction Gruppe an der Universität Bremen. Die Idee dieses Projekts war es, ein Smart-Home-System zu entwickeln, das greifbare Objekte als eine Methode zum Ausdruck von Emotionen verwendet. Es verwendet 3D-gedruckte Objekte mit RFID-Chips und einen RFID-Scanner. Je nach ausgewähltem Objekt ändert das System Musik und Beleuchtung.

Bachelor project at MARUM. The task was to autonomously detect pollution on beaches. It uses object detection to detect pollution. A pipeline with different pre- and post-processing methods was developed. (The report is only in German)

Bachelor Projekt (Meeresblick)

Bachelor-Projekt bei MARUM. Die Aufgabe bestand darin, Verschmutzungen an Stränden selbstständig zu erkennen. Zur Erkennung der Verschmutzung wird die Objekterkennung eingesetzt. Es wurde eine Pipeline mit verschiedenen Vor- und Nachbearbeitungsmethoden entwickelt.

Data Science course project

Data Science Master Course project. A segmentation model was developed for the purpose of detecting the dessication of lakes or coastal erosion. The model in question is known as the DeepLabV3+ model. The objective of the study was to detect water on RGB satellite images. The detection was then used to create a history of the amount of coastal erosion or dessication.

Data Science Kurs projekt

Data Science Masterkurs Projekt. Es wurde ein Segmentierungsmodell entwickelt, um die Austrocknung von Seen oder die Erosion von Küsten zu erkennen. Das betreffende Modell ist als DeepLabV3+-Modell bekannt. Ziel der Studie war es, Wasser auf RGB-Satellitenbildern zu erkennen. Die Erkennung wurde dann verwendet, um eine Historie des Ausmaßes der Küstenerosion oder Austrocknung zu erstellen.
